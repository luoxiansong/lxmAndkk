#Java |与|| 、&与&& ，或与非的使用和区别
一、“与”和“或”的定义区别
（1）使用“与”操作，要求所有表达式的判断结果都是TRUE才为真；若有一个为FALSE，那么最终判断结果则为FALSE。
（2）使用“或”操作，要求所有表达式的判断结果都是FALSE才为假；若有一个为TRUE，那么最终的结果则为TRUE。
